# window

#### window.open

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * window.open(path: string, target: string, Configuration);
 * 
 * 
 * @path: 打开文件的路径
 * @target: 打开方式
 * @Configuration: 窗口配置信息
 * 
 * 打开一个新页面
 */

window.open('http:www.baidu.com', null, 'width=300, height=300');


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
