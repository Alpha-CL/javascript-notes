# Math Methods

## Important Methods

#### Math.random();

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 返回0到1之间的伪随机数
 * 
 */

random(): number;


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


// Math.random()与使用Math.floor()可用于返回随机整数

Math.floor(Math.random() * 10);             // returns a random integer from 0 to 9

Math.floor(Math.random() * 11);             // returns a random integer from 0 to 10


Math.floor(Math.random() * 100);            // returns a random integer from 0 to 99

Math.floor(Math.random() * 101);            // returns a random integer from 0 to 100

Math.floor(Math.random() * 10) + 1;         // returns a random integer from 1 to 10

Math.floor(Math.random() * 100) + 1;        // returns a random integer from 1 to 100


// [12, 36)

Math.random() * 24;                         // [0, 24)

12 + Math.random() * 24;                    // [12, 36)                    

Math.floor(12 +  1 + Math.random() * 24);   // [12, 36)


//-------------------------------------------------------------------------------------------------------------------//


// 抽奖概率

var num = Math.random() * 100;

if (num <= 1) {

    console.log('一等奖');
    
} else if (num <= 2)  {

    console.log('二等奖');

} else if (num <= 3) {

    console.log('三等奖');
    
} else {

    console.log('谢谢回顾');
}


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### Math Properties (Constants)

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


Math.E        // returns 电子属性欧拉数和自然对数的基数，约等于2.718
Math.PI       // returns 的PI属性返回的圆的面积的比例到其半径的平方，大约3.14
Math.SQRT2    // returns 所述SQRT2属性返回的2的平方根，约1.414
Math.SQRT1_2  // returns 所述SQRT1_2属性返回的1/2的平方根，约0.707
Math.LN2      // returns 在LN2属性返回2的自然对数，约为0.693
Math.LN10     // returns 该LN10属性返回10的自然对数，约为2.302
Math.LOG2E    // returns 所述LOG2E属性返回E的基数为2的对数，约1.442
Math.LOG10E   // returns 所述LOG10E属性返回E的10为底的对数，约0.434


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
