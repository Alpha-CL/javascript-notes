## Append

#### appendChild

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 添加一个节点作为节点集合的最后一个子节点
 * 
 * 类似剪切，如果是把页面中已有的元素添加到另一个元素中，则之前的添加后就没有了
 */

appendChild<T extends Node>(newChild: T): T;


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
