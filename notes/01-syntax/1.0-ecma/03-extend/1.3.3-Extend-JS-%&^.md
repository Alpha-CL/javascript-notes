# Bit Operation

将一个整数的二进制格式进行计算

JS中，如果对一个数据进行位运算，它首先会将其转换为一个整数，并且按照 32 位置的整数二进制排列


``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


                      |                       |                             
                   ^  |  %                % ^ |                                
                   |  |  |                 |  |                              
                   -  |  -                 -  |                              
        -----|-----|--|--|-----|-----|-----|--|--|-----|-----|---------->
                      |                       |   
            ...   -3  | -2    -1     0     1  |  2     3     ...                                    
                      |                       |                                          
                      |                       |                                          

        %        -2.5 =>  2               2.5 => 2
        ^        -2.5 => -3               2.5 => 2
        
        /**
         * %
         * 
         * 趋近于零: 直接省略整数后的小数
         */                                                                                
                                                                                        
        /**
         * ^
         * 
         * 向下取整: 取较小方向第一个整数
         */                                                                               


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


/**
 * 求余: %                    // rem
 * 
 * 求余的正负号仅与 '被除数' 有关
 */

x rem y: x - n * y;

n = x / y = float ≈ int;


//-------------------------------------------------------------------------------------------------------------------//


/**
 * 求模: ^                    // mod
 * 
 * 求模的正负号仅与 '除数' 有关
 */

x mod y: x - n * y;

n = x / y = -float ≈ floor(-float);


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
