#### echarts api

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 
 * 
 * 
 * 
 */


echarts                 // 初始化 echartsInstance 的各种方法

echartsInstance         // 配置 echartsInstance(options)

action                  // 事件

events                  // 


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


1) 新建 echart 实例

    const echart = new Echarts(config);             // 创建 图表实例


2) 配置 echart

    echartsInstance.setOption(option);              // 可以依次分配多个事件
    
    echartsInstance.on(event);                      // 绑定事件处理函数
    echartsInstance.off(event);                     // 解绑事件处理函数 


3) 绑定事件

echartsInstance.dispatchAction();                   // 


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
