#### global variable

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 释放变量时的副作用


使用 var 创建的全局变量( 在函数外部创建的 )                      // 不能删除
不使用 var 创建的隐含全局变量( 尽管它是在哈数内部创建 )            // 可以删除


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


隐含全局变量严格来讲不是真正的变量, 而是全局对象的属性
属性可以通过 delete 删除, 但变量不可以



var global_var = 1;

global_novar = 2;
(function() {
    global_fromfunc = 3;
});


delete global_var;              // false            // number
delete global_novar;            // true             // undefined
delete global_fromfunc;         // true             // undefined


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### don't extend the built-in prototype

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 不要在内置原型上扩展方法


1) 严重影响可维护性( 会造成不可预测的不可控风险 )

2) 给原型添加的属性在没有使用 hasOwnProperty(); 时，可能会在 循环中出现( 导致使用混乱 )


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


// 允许谨慎添加属性


1) 使用 js 即将定义的内置成员( 等待浏览器对其支持 )

2) 不同浏览器可能有的支持有的未支持, 则可以扩展未支持浏览器上该方法

3) 准确的定义并书写其方法的文档, 并与团队交流


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### configuration object pattern

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 当函数参数过多时, 使用配置对象


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


var conf = {
    name: 'alpha',
    age: 18,
    gender: 'male'
};


addPerson(conf);


// 优点

    - 不需要记住众多的参数及其顺序
    
    - 可以安全忽略可选参数
    
    - 更加易于阅读和维护
    
    - 更加易于添加和删除参数


// 缺点

    - 需要记住参数的名称
    
    - 属性名称无法被压缩


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
